闭包

# 概念

闭包就是能够读取其他函数内部变量的函数。

  • 闭包的形式：函数内部定义函数
  • 本质上闭包就是将函数内部和外部连接起来的一座桥梁

# 作用

闭包的作用：

  • 可以读取函数内部变量
  • 让这些变量始终保持在内存中，即闭包可以使得它诞生的环境一直存在。
  • 封装对象的私有属性和私有方法

# 代码实现

内访外：

function fn() {
        var num = 100
        function fun() {
          console.log(num)
        }
        fun()
      }
fn()
// 100

# 闭包案例

# 1.循环注册点击事件

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>循环注册点击事件.</title>
    <style>
      .nav {
        cursor: pointer;
      }
    </style>
  </head>
  <body>
    <ul class="nav">
      <li>晴天</li>
      <li>七里香</li>
      <li>一路向北</li>
      <li>本草纲目</li>
    </ul>

    <script>
      var lis = document.querySelector('.nav').querySelectorAll('li')

      for (var i = 0; i < lis.length; i++) {
        (function (i) {
          lis[i].onclick = function () {
            console.log(lis[i].innerHTML)
          }
        })(i)
      }
    </script>
  </body>
</html>
